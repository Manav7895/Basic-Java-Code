Primitive Data Types
Question 1 - Write a program to declare and initialize variables of all primitive data types (int, long, float, double, char, boolean, etc.) and print their values.

If-Else Statement
Question 2 - Write a program that takes an integer input and checks whether it is positive, negative, or zero using an if-else statement.

If-Else-If Ladder
Question 3- Write a program to take marks as input and print the grade based on the following criteria:

Marks >= 90: Grade A
Marks >= 80: Grade B
Marks >= 70: Grade C
Marks < 70: Fail

Nested If-Else
Question 4- Write a program that takes a number as input and checks if it is even. If the number is even, check if it is also divisible by 4.

Switch Statement
Question 5 - Write a program that takes a day number (1 for Monday, 2 for Tuesday, etc.) and prints the corresponding day name using a switch statement.
